Explicação

O raciocínio parte da observação de padrões repetidos em várias turmas para sugerir uma conclusão provável. Acumular casos para propor uma regra geral é a marca do raciocínio indutivo.

ESTUDO DE CASO

O raciocínio parte da observação de padrões repetidos em várias turmas para sugerir uma conclusão provável. Acumular casos para propor uma regra geral é a marca do raciocínio indutivo.

Dica: O raciocínio indutivo vai do particular (vários exemplos específicos observados) para o geral (uma regra ou probabilidade). Ele não garante 100% de certeza, mas indica uma forte probabilidade.
